Pet project for https://stepik.org/course/4852/syllabus

# Прогнозирование успешного завершения онлайн-курса

## Описание проекта
Прогнозирование успешного завершения курса "Анализ данных в R" на основе данных о первых двух днях активности студентов.

## Задача
Предсказать, наберет ли пользователь более 40 правильных решений за курс, используя только данные о первых двух днях активности.

## Данные
- `event_data_train.csv` - события пользователей (тренировочные)
- `submissions_data_train.csv` - решения пользователей (тренировочные)
- `events_data_test.csv` - события пользователей (тестовые)
- `submission_data_test.csv` - решения пользователей (тестовые)
